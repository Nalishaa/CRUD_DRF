# Grocery Bud

A CRUD grocery list app built with Django REST Framework and React.

## Features

- Add, edit, and delete grocery items
- Check off items with a strikethrough
- Toast notifications for actions
- Responsive mobile layout

## Tech Stack

- Django REST Framework
- React + Vite
- SQLite
- CSS

## Deployment

- **Frontend:** Netlify
- **Backend:** Railway
